## 🐛 故障排除

### 常見問題

#### 1. Pico 無法連接 WiFi
```python
# 檢查配置是否正確
print(f"SSID: {WIFI_SSID}")
print(f"Password: {WIFI_PASSWORD}")
```

#### 2. MQTT 連接失敗
```bash
# 檢查 Broker 是否運行
sudo systemctl status mosquitto
```

#### 3. 資料庫連接失敗
```bash
# 檢查 MongoDB 是否運行
docker ps | grep mongo
```

#### 4. API 無法訪問
```bash
# 檢查服務是否運行
ps aux | grep python
```
