# 能源監控系統

## 專題簡介

模擬家庭能源監控系統，使用溫度變化模擬用電量，監測異常用電並提供統計分析，幫助節能減碳。

## 功能列表

- [x] 即時監測用電量（溫度模擬）
- [x] 異常用電警報
- [x] 每日/每月用電統計
- [x] 用電趨勢分析
- [x] 節能建議
- [x] 用電排行榜

## 系統架構

```
Pico (用電監測)
  ├─ 感測器（模擬電表）
  └─ LED（異常警報）
  ↓ MQTT
Pi (資料分析)
  ├─ MongoDB（資料儲存）
  ├─ 分析引擎（統計分析）
  └─ FastAPI（API 服務）
```

## 創新特色

1. **智慧分析**
   - 機器學習預測用電
   - 異常檢測演算法
   - 個人化建議

2. **視覺化報表**
   - 用電趨勢圖表
   - 比較分析
   - 節能成效追蹤

3. **遊戲化設計**
   - 節能積分系統
   - 排行榜競賽
   - 成就徽章

## 主要功能

### 異常檢測
```python
def detect_anomaly(current, history):
    avg = sum(history) / len(history)
    if current > avg * 1.5:
        return True, "用電異常偏高"
    return False, "正常"
```

### 統計分析
```bash
# 查看今日用電
curl http://localhost:8000/api/energy/today

# 查看本月統計
curl http://localhost:8000/api/energy/month

# 查看節能建議
curl http://localhost:8000/api/energy/suggestions
```
