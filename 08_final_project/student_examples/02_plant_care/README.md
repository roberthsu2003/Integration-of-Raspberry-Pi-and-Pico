# 智慧植物照護系統

## 專題簡介

這是一個智慧植物照護系統，能夠監測環境溫度（模擬土壤濕度），並在需要時提醒澆水（LED 閃爍）。系統記錄植物生長環境資料，幫助分析最佳照護條件。

## 功能列表

- [x] 監測環境溫度（模擬土壤濕度）
- [x] 資料透過 MQTT 傳送
- [x] 自動儲存到資料庫
- [x] 需要澆水時 LED 閃爍提醒
- [x] 記錄澆水歷史
- [x] 提供生長統計分析

## 系統架構

```
Pico (環境監測)
  ├─ 溫度感測器（模擬濕度）
  └─ LED（澆水提醒）
  ↓ MQTT
Pi (資料處理)
  ├─ MongoDB（資料儲存）
  ├─ FastAPI（API 服務）
  └─ 分析引擎（統計分析）
```

## 創新特色

1. **智慧提醒系統**
   - 根據歷史資料預測澆水時間
   - 避免過度澆水
   - 考慮環境因素

2. **生長記錄**
   - 記錄每日環境資料
   - 生成生長曲線
   - 提供照護建議

3. **多植物支援**
   - 支援監測多盆植物
   - 不同植物不同閾值
   - 統一管理介面

## 使用方法

### 設定閾值
```python
# config.py
MOISTURE_LOW = 30  # 低濕度閾值（%）
MOISTURE_HIGH = 70  # 高濕度閾值（%）
CHECK_INTERVAL = 300  # 檢查間隔（秒）
```

### 查詢統計
```bash
# 查看植物狀態
curl http://localhost:8000/api/plant/status

# 查看澆水歷史
curl http://localhost:8000/api/plant/watering-history

# 查看生長統計
curl http://localhost:8000/api/plant/statistics
```
