# 貢獻指南

感謝你對本課程專案的興趣！

## 如何貢獻

### 回報問題
- 使用 GitHub Issues 回報錯誤
- 提供詳細的錯誤訊息和重現步驟
- 說明你的環境（OS、Python 版本等）

### 建議改進
- 提出新功能建議
- 改進現有文件
- 分享教學經驗

### 提交程式碼
1. Fork 本專案
2. 建立功能分支
3. 提交變更
4. 發送 Pull Request

## 程式碼規範

### Python
- 遵循 PEP 8
- 加入中文註解
- 撰寫文件字串

### 文件
- 使用清晰的中文
- 提供範例
- 加入檢核清單

## 測試

提交前請測試：
- 程式碼能正常執行
- 文件連結有效
- 範例可以運作

## 授權
