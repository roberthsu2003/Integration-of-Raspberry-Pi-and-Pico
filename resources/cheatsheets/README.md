# 速查表

本目錄包含課程中使用的主要技術的快速參考指南。

## 📚 可用的速查表

### 1. [MicroPython 速查表](micropython_cheatsheet.md)

快速參考 Raspberry Pi Pico 常用的 MicroPython 功能。

**涵蓋內容：**
- 基礎語法和模組匯入
- GPIO 控制（數位輸入/輸出、中斷）
- 類比輸入（ADC、溫度感測器）
- PWM（脈衝寬度調變）
- 計時器
- 通訊協定（UART、I2C、SPI）
- WiFi 連接（Pico W）
- 檔案系統操作
- JSON 處理
- 錯誤處理
- 實用技巧和常見問題

**適用於：**
- Day 1-2: Pico 基礎操作
- Day 4-5: MQTT 通訊（Pico 端）
- Day 5: UART/USB 通訊
- 所有涉及 Pico 程式設計的課程

---

### 2. [FastAPI 速查表](fastapi_cheatsheet.md)

快速參考 FastAPI 常用功能和模式。

**涵蓋內容：**
- 基礎設定和啟動
- 路由定義（GET、POST、PUT、DELETE）
- 資料模型（Pydantic）
- 回應模型和狀態碼
- 錯誤處理
- 依賴注入
- 中介軟體（CORS 等）
- 背景任務
- 檔案上傳
- 靜態檔案和模板
- WebSocket
- 資料庫整合（MongoDB）
- 測試
- 設定管理
- 日誌
- 安全性

**適用於：**
- Day 3: Pi 基礎與資料庫
- Day 6: Pi-Pico 整合應用
- Day 7: 多裝置管理
- Day 8: 範例專案
- 所有涉及 API 開發的課程

---

### 3. [MQTT 速查表](mqtt_cheatsheet.md)

快速參考 MQTT 協定和 Paho MQTT 客戶端的常用功能。

**涵蓋內容：**
- MQTT 基礎概念（Broker、Publisher、Subscriber、QoS）

- Python Paho MQTT 客戶端

- 主題命名規範和萬用字元
- 訊息格式（JSON、純文字）
- 進階功能（遺囑訊息、保留訊息、認證、TLS）
- 錯誤處理和重連機制
- 測試工具
- 效能優化
- 常見問題

**適用於：**
- Day 4-5: MQTT 通訊
- Day 6: Pi-Pico 整合應用
- Day 7: 多裝置管理
- 所有涉及 MQTT 通訊的課程

## 🎯 如何使用這些速查表

### 學習階段
1. **課前預習**：快速瀏覽相關速查表，了解將要學習的內容
2. **課中參考**：遇到不確定的語法時，快速查找
3. **課後複習**：鞏固學習內容，查漏補缺

### 實作階段
1. **快速查詢**：不需要記住所有語法，需要時查閱即可
2. **範例參考**：速查表中的範例可以直接複製使用
3. **除錯輔助**：遇到問題時，檢查語法是否正確

### 專題開發
1. **設計參考**：規劃專題時，查看可用的功能
2. **實作指南**：按照速查表的範例實作功能
3. **優化提示**：參考進階功能和最佳實踐

## 🎓 學習建議

### 不要死記硬背
- 速查表的目的是快速參考，不是要你全部記住
- 理解概念比記住語法更重要
- 多實作，自然就會記住常用的部分

### 動手實踐
- 看到範例就試著執行
- 修改範例，看看會發生什麼
- 遇到問題就查閱速查表

### 循序漸進
- 先掌握基礎功能
- 再學習進階功能
- 最後探索優化技巧

### 建立連結
- 將不同技術結合使用
- 理解它們之間的關係
- 建立完整的知識體系

## ❓ 常見問題

### Q: 速查表太長，如何快速找到需要的內容？
A: 使用編輯器的搜尋功能（Ctrl+F 或 Cmd+F），輸入關鍵字快速定位。

### Q: 速查表的範例可以直接使用嗎？
A: 可以！所有範例都經過測試，可以直接複製使用。但建議理解後再使用。

### Q: 如何離線使用速查表？
A: 速查表都是 Markdown 格式，可以：
- 在任何文字編輯器中開啟
- 轉換成 PDF 列印
- 使用 Markdown 閱讀器離線查看

### Q: 速查表涵蓋所有內容嗎？
A: 速查表涵蓋常用功能，但不是完整文件。深入學習請參考官方文件。
