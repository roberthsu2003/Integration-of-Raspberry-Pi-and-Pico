# 更新日誌

本文件記錄專案的所有重要變更。

## [1.0.0] - 首次發布

### 🎉 完整的物聯網整合課程

完整的 Raspberry Pi 和 Pico 物聯網整合課程講義，包含 9 個模組、35+ 程式範例、完整文件和輔助工具。

### ✨ 課程模組

#### 基礎模組
- **Pico 基礎** - LED 控制、感測器讀取、按鈕輸入（11 個程式範例）
- **Pi 基礎** - Docker、MongoDB、FastAPI、RESTful API
- **MQTT 通訊** - Broker 設定、Pico 發布者、Pi 訂閱者、測試工具
- **UART/USB 通訊** - 串列通訊基礎和範例

#### 進階模組
- **整合應用** - 完整的資料收集系統、端到端測試
- **多裝置管理** - 裝置管理系統、狀態監控、儀表板
- **範例專案** - 5 個完整的實用專案範例
- **綜合專題** - 專題模板、學生範例、評量標準

### 🛠️ 輔助工具

- `verify_setup.py` - 環境驗證工具
- `test_mqtt.py` - MQTT 連接測試
- `check_api.py` - API 服務檢查

### 📚 學習資源

- MicroPython、FastAPI、MQTT 速查表
- 教師指引文件
- 故障排除指南
- 參考資源整理

### 📖 完整文件

- 專案 README 和環境設定指南
- 詳細課程時間表
- 貢獻指南和授權條款
- 課程分發指南和回饋機制

### 📊 專案統計

- **程式碼檔案：** 35+
- **文件檔案：** 30+
- **總檔案數：** 65+
- **程式碼行數：** 5,000+
- **文件字數：** 35,000+
